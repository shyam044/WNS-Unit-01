2) Wireless Security Industry

i) WEP

  • WEP stands for Wired Equivalent Privacy
  • Most Wi-Fi networks used WEP at that time
  • WEP uses RC4 Encryption Algorithm
  • Attackers can easily launch attacks on the system

ii) Hacking Tools

  • Hacking Tools like Air-Snort, Air-Crack, Air-Crack-ng are used to break WEP Encryption
  • Even normal people can launch attacks by using these hacking tools

iii) Poor Security

  • Manufacturers set up a Wi-Fi Network, and they didn't focus much on security
  • Most users don't know the risks of using an insecure wireless network

iv) WPA to WPA2

  • WPA stands for Wi-Fi Protected Access
  • WPA was introduced to fix WEP vulnerabilities
  • Later, WPA2 came out and offered stronger encryption and security

v) Awareness

  • Activities like wardriving give awareness of using insecure wireless networks

3) Communication Jamming

  • Communication jamming disrupts the signal and blocks the communication between sender and receiver
  • Communication jamming is an Active attack
  • It is difficult to detect the attack

Types of Communication Jamming:

i) Denial of Service Jamming

  • Denial of Service Jamming focuses on jamming an entire network
  • It blocks the communication between client and Base Station
  • In Denial of Service Jamming, Darth disrupts the server and prevents the services to server

ii) Client Jamming

  • Client Jamming focuses on jamming a client station
  • It blocks the communication between client and Application
  • A rogue client can act as the legitimate (real) one

iii) Base Station Jamming

  • Base Station Jamming focuses on jamming a Base station
  • It blocks the radio link between client and Base Station
  • A rogue Base Station can act as the legitimate (real) one

4) Rogue Network Access Point

  • Rogue Network Access Point is a fake Network device that acts as the legitimate (real) one
  • Rogue Network Access Point can be a Passive attack or an Active attack
  • It is easy to detect the attack when compared to the communication jamming

5) Operations of Rogue Client and DoS Jamming

Operations of Rogue Client:

i) Information gathering

  • The attacker gathers the information by identifying:
    • SSIDs,
    • MAC addresses,
    • and traffic patterns.

ii) Cloning or Device Stealing

  • The attacker clones the client identity or steals the physical device to access the login information

iii) Fake Client Connection

  • The Rogue client uses a cloned identity to connect to the network
  • The Network allow access for connection because the cloned identity looks same as a valid client identity

iv) Unauthorized activity

  • Once the Rogue client connects to the Network, it launches security attacks to the Network

Operations of DoS Jamming:

i) Information gathering

  • The attacker gathers the information about:
    • target frequency/channel
    • coverage area
    • traffic patterns

ii) RF injection

  • The Attacker injects a strong radio frequency into the target channel

iii) Acknowledgement Failure

  • Increasing the Radio Frequency signal, the receiver can't decode the packet and make Acknowledgement and handshake fails.

iv) Continuous Jamming

  • The attacker keeps the network unusable by continuously jamming in that particular area

How it works in wireless network?

Position & Lure:

  • The attacker creates a fake WiFi network.
  • They trick people into connecting to this fake network instead of the real one.

Intercept:

  • When people connect to the fake WiFi network, all their information is passed to the attacker's device.

Relay & Manipulate:

  • User don't realize that attacker is there in between the communication of user and Internet
  • attacker can change the data by inserting harmful code

Unsecure website and Apps:

  • Attacker steals the data from unsecure website and App.
    • Ex: stealing login details from user
  • Attacker uses this stolen data to login to accounts.

Example:

The attacker created a fake WiFi network in Railway station.
They trick the people to connect to this network and all their internet activity is passed to the attacker's device.
Whenever user visit the Banking site, attacker captures the login-details and change the data by inserting harmful code.

7) Importance of Uncontrolled Terrain in Wireless Security

  • Wireless signal travels through air, not through wires
  • The signal spreads in all directions
  • Anyone can easily attack the network

i) Easy to attack

  • A wired network requires physical access to attack the network
  • A wireless network doesn't require physical access. They easily attack the network at the shortest distance, or by using powerful antennas

ii) Communication Jamming

  • Communication jamming disrupts the signal and blocks the communication between sender and receiver
  • Communication jamming is an Active attack
  • It is difficult to detect the attack
  • Types of Communication Jamming:
    • a) Denial of Service Jamming
    • b) Client Jamming
    • c) Base Station Jamming

iii) Rogue Network Access Point

  • Rogue Network Access Point is a fake Network device that acts as the legitimate (real) one
  • Rogue Network Access Point can be a Passive attack or an Active attack
  • It is easy to detect the attack when compared to the communication jamming

iv) Eavesdropping

  • Wireless network signal spreads in all directions
  • Anyone can easily attack the network
  • If data is not encrypted, attackers can easily read the data
